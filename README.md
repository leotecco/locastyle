# Locastyle

This gem provides the front-end framework [Locastyle](https://github.com/locaweb/locawebstyle), at version **3.9.0**, for your ruby application.

## Dependencies

#### JQuery

Recommended use jQuery 1.10 or up

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'locastyle'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install locastyle

## Usage

#### Include locastyle javascript assets

Add the following to your `app/assets/javascripts/application.js`:

    //= require locastyle

#### Include locastyle stylesheet assets

Add to your `app/assets/stylesheets/application.css`:

    *= require locastyle
